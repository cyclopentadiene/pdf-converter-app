Features

Everything you need to turn PDFs into usable Markdown, built into a native desktop app.

One-click conversion

Select a PDF, choose where to save, done. No configuration, no command-line flags, no complexity.

Page-preserving

Each PDF page becomes a separate Markdown section, maintaining the logical structure of your document.

Image extraction

Embedded images are saved to a companion folder alongside your Markdown output, with references intact.

Table support

Tables in your PDF are detected and converted to proper Markdown table syntax, ready for any renderer.

Native desktop GUI

Runs as a standalone macOS app. No browser, no Electron, no runtime dependencies for end users.

Getting Started

Up and running in under a minute with uv.

  1. Install dependencies

    Use uv to synchronize the project environment and install all required packages.

  2. Run the app

    Launch the graphical interface with a single command. Select your PDF and choose an output location.

  3. Build for distribution (optional)

    Package as a standalone macOS application using PyInstaller. No Python install required on the target machine.

# Install dependencies
uv sync

# Run the app
python pdf2md.py

# Build standalone macOS app (optional)
uv sync --extra build
pyinstaller "PDF Converter.spec"
